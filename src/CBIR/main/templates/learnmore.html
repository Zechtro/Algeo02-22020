Search Engine Concept

Database

Gambar yang di-upload akan dimasukkan ke dalam database terlebih dahulu. Database untuk gambar yang ingin dilakukan pencarian berbeda dengan database untuk dataset.Database untuk gambar masukan memiliki field image (nama file gambar), result (jumlah gambar pada dataset yang memiliki kemiripan lebih besar dari 60% terhadap gambar masukan), dan search_time (waktu proses perhitungan tingkat kemiripan). Sedangkan database untuk dataset memiliki field image (nama file gambar) dan similarity (tingkat kemiripan dengan gambar masukan dengan skala 0-100).

Request-Response

Gambar yang telah diupload kemudian dimasukkan ke dalam database dengan nilai pada atribut image adalah nama file gambar tersebut. Sedangkan atribut gambar masukan lainnya dimasukkan dengan nilai default yaitu nol. Untuk atribut gambar dataset similarity dibiarkan bernilai NULL terlebih dahulu. Nilai similarity tersebut kemudian akan diperbarui ketika proses perhitungan dijalankan. Setelah didapatkan seluruh tingkat kemiripan dataset terhadap gambar masukan, kemudian gambar pada dataset yang memiliki tingkat kemiripan lebih dari 60% akan ditampilkan pada website.
